hexo基本用法

发表于2024-05-04|更新于2024-05-06|hexo的用法
|阅读量:

  1. butterfly的官网，有各种官方文档：Butterfly - A Simple and Card UI Design theme for Hexo

  2. hexo需要下载安装，需要用到node.js，npm。需要在GitHub或gitee上面创建好仓库

  3. hexo+butterfly的基本命令

    1. hexo new page “标题名”,生成以后，需要加入type类型，自定
    2. hexo s 生成页面

  4. hexo d 上传到GitHub仓库（需要先git连接仓库）

  5. 如何添加文章：在_posts文件夹下添加文件夹，写入自己要更新的文章即可
